uint8_t sdmmc_changeBlockLength ( uint16_t  pLength)

Changes the length of a block in read and write operations.

This command utilizes CMD16 to alter the block size which will be used in read and write commands. Please note that this feature might not work with every SD card! If it fails, the default block size of 512 Bytes will remain unchanged.

Parameters:
pLengththe new length of a block. Set to 0 if default size should be set.
Returns:
TRUE on success, otherwise FALSE
