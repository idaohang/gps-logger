#define FREQUENCY   1

Indicates how many message packets per second shall be recorded Valid values for the ST22 are: [1, 2, 4, 8, 10]

#define LED_THRESHOLD   NUM_MESSAGES * FREQUENCY

The LED will blink every LED_THRESHOLD messages.

#define MESSAGES   GPS_NMEA_GGA | GPS_NMEA_RMC | GPS_NMEA_VTG

Indicates which messages to record per message packet Valid values for the ST22 are: [GPS_NMEA_GGA, GPS_NMEA_GSA, GPS_NMEA_GSV, GPS_NMEA_GLL, GPS_NMEA_RMC, GPS_NMEA_VTG, GPS_NMEA_ZDA]

int main ( void  )

Main method of the project.

This method contains the general workflow of the device. This is the place where the different modules are combined in order to acheive the actual logging of GPS-coordinates.

Returns:
Despite of being defined as a function returning an integer, the method will return nothing, as the code will stay in a while(TRUE)-loop prior to the return statement.
